# 오늘 뭐 먹지 (lunchpick)

단톡방에서 점심 메뉴를 30초 만에 정하는 그룹 투표 미니앱.

> 앱인토스 바이브코딩 챌린지(2026년 6월 · 주제: 일상이 편해지는 순간) 출품작.
> 앱인토스 WebView 미니앱(React/Vite). 빌드/실행/콘솔 등록값은 스캐폴드 후 본 README에 정리됩니다.

## 핵심 (MVP)
- 방 생성 / 후보 메뉴 등록
- 6자리 방 코드로 참여
- 1인 1표
- 실시간에 가까운 결과(폴링)
- 마감 후 1위 확정
